# מחשבון השקעה בדירה בפורטוגל 🏠

מחשבון אינטראקטיבי להערכת השקעה בדירה למגורים בפורטוגל, כתושב חוץ, עם משכנתא.
כל הנתונים ניתנים לעריכה והתוצאות מתעדכנות בזמן אמת.

## 🔗 לינק חי

לאחר הפעלת GitHub Pages, המחשבון זמין בכתובת שתופיע בהגדרות הריפו (Settings → Pages).

## מה המחשבון מחשב

- **עלויות רכישה** — מס רכישה IMT (מדורג / 7.5% אחיד לתושב חוץ), בולים (0.8% רכישה + 0.6% משכנתא + 4% על עמלות), נוטריון, עו"ד, עמלות בנק.
- **מימון** — תקופת משכנתא מוגבלת אוטומטית לפי גיל (כללי בנק פורטוגל + סיום עד גיל 75), אומדן ריבית לפי LTV/משך/סוג, ותמיכה בריבית קמפיין דו-שלבית.
- **מיסוי שכירות** — ארוך-טווח (קטגוריה F: 10% לשכר מתון עד €2,300/ח', או לפי משך חוזה) וקצר-טווח (AL: 35%×25% אפקטיבי, או 50% באזור עצור).
- **תזרים ותשואה** — תזרים חודשי נטו, Cash-on-Cash, תשואה ברוטו/נטו.
- **מודל יציאה** — שווי עתידי, יתרת משכנתא, מס רווח הון, ורווח כולל.

## הבהרה

זהו כלי הדמיה בלבד ואינו מהווה ייעוץ מס/משפטי/פיננסי. כל הנתונים מבוססים על מקורות רשמיים נכון ל-2026,
אך יש לאמת מול עו"ד ורו"ח בפורטוגל לפני התקשרות. שיעורי מס וריבית משתנים.

## הרצה מקומית

פשוט לפתוח את `index.html` בדפדפן — הכל עצמאי (ללא תלויות חיצוניות).
